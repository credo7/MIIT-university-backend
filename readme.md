Это **backend** часть для **образовательной платформы для логистов** ( она когда-то реализовывалась мною на TypeScript, но сейчас для себя хотел ее переписать на Python)
<br><br>
В main.py **мультипроцессор**, который вызывает **FastAPI сервер и Socket.IO**
<br><br>
**FastAPI** отвечает за создание Студентов, учителей, групп, их изменения и получения
<br><br>
**Socket.IO** отвечает за RealTime events:
* Подключенные компьютеры
* создание event для каждого компьютера
* чекпоинты по баллам
* финиш event’а
<br><br>
